기본적으로 지원하는 조사는 은/는, 이/가, 와/과, 을/를, 나/이나, 로/으로 입니다.

put( text, type, special )

단어에 맞는 조사를 적용한 결과를 반환

Parameters

  • text : 조사를 적용할 단어
  • type : 종성이 없을 때 조사(기본 조사는 종성 유무와 상관 없이 사용하고 special 생략)
  • special : 종성이 있을 때 조사(기본 조사 사용시 생략)

// 기본 지원 조사
postposition.put( "", "" )

결과 :

// 사용자 지정 조사
postposition.put( "", "", "" )
